# Simple-Linear-Regression-Project

Project Title
Salary Prediction Using Simple Linear Regression

What I Did
Collected and loaded the salary dataset into Google Colab.

Imported the dataset using Pandas.

Checked and cleaned the data for missing values.

Split the dataset into training and testing sets.

Built a Simple Linear Regression model to predict salary based on years of experience.

Trained the model using the training data.

Predicted salary values for the test data.

Evaluated the model using performance metrics.

Visualized the regression line and actual data points using graphs.

Interpreted the results and prepared the project report.

Technologies Used

Python
Google Colab
Pandas – Data loading and preprocessing
NumPy – Numerical operations
Matplotlib – Data visualization
Scikit-learn (sklearn) – Building and evaluating the Simple Linear Regression model

Model Used

Simple Linear Regression

Project Outcome
The model successfully predicted salary based on years of experience. This project helped me understand data preprocessing, model training, prediction, visualization, and basic machine learning concepts using Python.
